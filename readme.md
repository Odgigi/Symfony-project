# Entités

- User
    - email: string

    - password: string
    - pseudo: string

    - prix_plancher: float

    - prix_moyen: float
    - prix_maximum: float

    - img3: string

    + collec: Collec

- Note

    - note_pratique: integer

- Categorie

- Collec
